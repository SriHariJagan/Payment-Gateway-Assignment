# 💳 Payment Gateway UI

A modern, responsive, and secure payment gateway frontend built with Next.js, React, TypeScript, Tailwind CSS, Zustand, React Hook Form, and Zod.

---

# 🚀 Features

- ✅ Modern payment UI
- ✅ Real-time form validation
- ✅ Card type detection (Visa, Mastercard, Amex)
- ✅ Currency support (INR, USD, EUR)
- ✅ Beautiful responsive design
- ✅ Secure payment simulation
- ✅ Zustand state management
- ✅ React Hook Form integration
- ✅ Zod validation
- ✅ Payment status handling
- ✅ Transaction summary
- ✅ Animated loading states
- ✅ Professional Tailwind styling

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/SriHariJagan/Payment-Gateway-Assignment.git
```

---

## 2️⃣ Navigate To Project

```bash
cd Payment-Gateway-Assignment
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Start Development Server

```bash
npm run dev
```

# 🧠 Functionalities

## Card Formatting

- Auto formats card number
- Auto formats expiry date

Example:

```bash
4242424242424242
```

Becomes:

```bash
4242 4242 4242 4242
```
